# Ecommerce-Customer-Behavior-Analysis
##  Project Objective
To analyze customer behavior and identify patterns in purchasing, engagement, and segmentation to improve business decisions.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib & Seaborn
- Scikit-learn

## Key Analysis
- Customer spending patterns
- Impact of discounts
- Customer activity analysis
- Customer segmentation using K-Means

## Key Insights
- Majority of purchases occur without discounts, indicating strong product demand.
- High-value customers contribute significantly to revenue.
- A portion of customers are inactive and need re-engagement strategies.

## Machine Learning
Used K-Means clustering to segment customers into:
- High-value customers
- Low-value customers
- Inactive customers

## Business Recommendations
- Focus on retaining high-value customers
- Reduce unnecessary discounts
- Target inactive customers with marketing campaigns
